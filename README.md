# Paper submission for SysML 2019

## Requirements

- latex
- texlive-latex-extra
- texlive-fonts-recommended
- letexmk
- xzdec (required by tlmgr)

On Ubuntu:

``` shell
apt install -y \
  texlive-latex-base \
  texlive-latex-recommended \
  texlive-latex-extra \
  texlive-fonts-recommended \
  latexmk \
  xzdec
```
